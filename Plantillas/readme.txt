# Plantilla Básica de Componentes

Esta es una colección de componentes HTML que puedes usar en tus proyectos web. Cada componente está separado en archivos HTML individuales para facilitar la inclusión y reutilización.

## Estructura de Archivos
Componentes
  - `navbar.html`: Barra de navegación.
  - `titulo.html`: Título y descripción general.
  - `encuesta.html`: Formulario de encuesta.
  - `tarjeta.html`: Tarjeta de contenido.
  - `modal.html`: Ejemplo de modal.
  - `acordeon.html`: Secciones de acordeón.
  - `tabs.html`: Navegación por pestañas.
  - `botones.html`: Botones de acción.
estilos:
-archivo css con todos los estilos y diseños
